## 🌟 Основные возможности

- 🔄 **Классические правила Конвея** — Моделирование эволюции колоний клеток в реальном времени.
- 📐 **Настраиваемое поле** — Задание произвольной ширины и высоты игрового пространства.
- 📂 **Загрузка конфигураций** — Чтение исходных состояний (паттернов, планеров, осцилляторов) из текстовых файлов *.txt.
- ⚡ **Высокая производительность** — Эффективное обновление матрицы состояния клеток.

## 🛠 Требования и Сборка

### Требования:
- Компилятор C++17 (GCC, Clang или MSVC)
- CMake 3.16+

### Сборка через CMake:
`ash
git clone https://github.com/Vane4ka2k2/GameOfLife.git
cd GameOfLife
cmake -B build
cmake --build build
`

---

## 🚀 Использование

После сборки запустите исполняемый файл:

`ash
# Windows
.\build\Debug\GameOfLife.exe

# Linux / macOS
./build/GameOfLife
`

По умолчанию приложение может загрузить стартовую карту из файла oard.txt в рабочей директории.
