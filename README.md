# GameOfLife

Консольная реализация знаменитого клеточного автомата «Игра «Жизнь»» (Conway's Game of Life) на языке C++.

## 🎮 Суть игры
Игра моделирует эволюцию колонии живых клеток на двумерной сетке по следующим правилам:
1. **Выживание**: Любая живая клетка с двумя или тремя живыми соседями выживает в следующем поколении.
2. **Гибель**: Живая клетка с менее чем двумя (одиночество) или более чем тремя (перенаселенность) соседями погибает.
3. **Рождение**: В пустой (мертвой) клетке, рядом с которой ровно три живые клетки, зарождается жизнь.

## 🛠️ Особенности реализации
Проект предлагает удобное текстовое меню выбора начального состояния игрового поля:
1. **Стандартное поле**: Поле размером 10x10 клеток, заполненное случайным образом.
2. **Квадратное поле N x N**: Настройка размера стороны поля с автоматическим случайным заполнением клеток.
3. **Прямоугольное поле M x N**: Индивидуальная настройка количества строк и столбцов со случайным заполнением.
4. **Загрузка из файла**: Загрузка начальной конфигурации доски из файла `board.txt` (где `1` обозначает живую клетку, а `0` — мертвую).

## 🚀 Требования и технологии
- **Язык**: C++ (поддержка C++11 и выше).
- **Платформа**: Разработано под Windows (использует заголовочный файл `<Windows.h>` для очистки экрана и управления задержками анимации).
- **Среда разработки**: Проект содержит готовые файлы решений Visual Studio (`.sln`, `.vcxproj`).

## 📦 Инструкция по сборке и запуску

### 1. Сборка через Visual Studio:
Откройте файл `GameOfLife.sln` в Visual Studio и соберите решение (Build -> Build Solution) в конфигурации **Release** или **Debug**.

### 2. Сборка через командную строку (MSVC):
Скомпилируйте исходные файлы с помощью компилятора `cl`:
```cmd
cl /EHsc main.cpp GameOfLife.cpp
```

### Формат файла `board.txt` для загрузки кастомной доски:
Пример содержимого файла (сохраните его рядом с исполняемым файлом как `board.txt`):
```text
3 3
0 1 0
0 1 0
0 1 0
```
*(Первая строка задает размеры: `строки` и `столбцы`, далее следует само игровое поле).*
